系統性回顧的
九個關卡 A Skill Map for Systematic Review & Meta-Analysis

從研究空缺確認、文獻搜尋、AI 篩選、PRISMA 流程、統計合併，到投稿前的最後稽核 — 帳號內這套 skill 把整條 SR／統合分析的工作鏈串成可逐關推進的流程。

9 個研究 skill 1 個總調度入口 5 階段強制順序（C1→C5） 4 個輔助產出工具
0
Orchestrator · 總調度

流程導航器

入口

不確定從哪一步開始時，先進這裡。它會判斷目前所在的研究階段、給出對應行動建議，並分流呼叫下方各子 skill。

meta-analysis-workflow

SR／統合分析全流程的總入口與導航器，輸出整體 checklist 與進度追蹤表。

我要做 meta 分析 SR 怎麼開始 幫我做系統性回顧 不知道從哪裡開始做 SR
1
Gap Check · 前置查核

研究空缺確認

動手搜尋前，先確認這個主題有沒有人做過、有沒有正在進行的試驗，避免重工並找出 research gap。

cochrane-trials-check

查核 Cochrane Library 有無既有 SR／Protocol，並透過 ClinicalTrials.gov v2 REST API 取得已完成或進行中 RCT 的結構化資料（NCT 編號、期別、樣本數、主要終點），產出研究空缺確認報告。

有沒有 Cochrane 查臨床試驗登記 research gap 確認 設計研究前查一下
2
Search · 文獻搜尋

多源文獻檢索

英文走 PubMed／PMC，中文走 CNKI；兩者都會把關鍵字拆解成多角度查詢策略，再去重整理成 PRISMA 友善的結構化報告。

pubmed-research-search

優先使用官方 PubMed MCP 工具取得真實 PMID、完整摘要與 PMC 全文，依 PICO／MeSH 分機制研究／系統性回顧／臨床試驗三組查詢，可經 CMU proxy 下載訂閱期刊全文。

搜 PubMed 找 RCT 找 meta-analysis 下載 PMC 全文
cnki-research-search

搜尋知網真實收錄的中文期刊論文（含 CNKI 連結／DOI），中醫藥、中國本土研究等領域優先使用，可選擇性透過校園 VPN 下載 CAJ/PDF。

搜知網 找中文文獻 CNKI 搜尋 查中醫藥研究
3
Screening · 篩選

AI 輔助文獻篩選

搜回來上百篇，用主動學習（active learning）大幅減少 title／abstract 篩選工時。

asreview-screening

協助使用 ASReview LAB 進行 AI 輔助篩選：安裝指引、資料格式準備（CSV／RIS／XLSX）、操作 SOP，匯出 included／excluded 清單供 PRISMA 使用。

幫我篩文獻 文獻太多怎麼篩 active learning 篩選 我有幾百篇要篩
4
Flow Diagram · 流程圖

PRISMA 篩選流程圖

把篩選各階段數字記錄下來，產出可直接嵌入論文的標準流程圖。

prisma-flow

透過互動問答收集篩選數字，產生符合 PRISMA 2020 的 SVG 向量圖與 Markdown 摘要，支援「僅資料庫搜尋」與「含其他來源」兩種版本，並可產出篩選數字核對表。

PRISMA 圖 幫我畫篩選流程 PRISMA 2020 文獻篩選圖
5
Statistics · 統計合併

統合分析運算

用 R 套件 meta 做定量合併，輸出圖表、結果解讀與論文撰寫用描述文字。

r-meta-analysis

涵蓋資料格式準備、函數選擇（metacont／metabin／metagen／metacor／metarate）、隨機效應模型（REML）、forest plot、異質性（I²／τ²／Q）、subgroup、meta-regression、leave-one-out 敏感度分析，以及 funnel plot ＋ Egger's test ＋ trim-and-fill 發表偏誤檢定。

跑 meta 分析 forest plot Egger's test subgroup analysis leave-one-out
6
Citation Check · 引用核對

文獻引用核對

寫完 Introduction／Discussion 後，逐筆核對引用是否真實、相關、年份作者正確。

academic-reference-checker

核對引用文獻的作者名、年份與內容相關性，搜尋未提供原文的引用，建議替換低相關性文獻；輸出中文核對報告、英文修正版段落，以及可匯入 EndNote／Zotero 的清單。

確認引用 核對文獻 verify citations 這段引用對嗎
7
Final Audit · 投稿前防線

稿件嚴謹修稿

投稿前最後一關，依 C1→C5 強制順序執行，不允許跳步或並行。硬性限制：C1 未完成不得寫 final dataset，C3 未完成不得潤稿結果段。

manuscript-review

把整份投稿包做 cross-file 版本一致性稽核，確保所有下游數字都能追溯到鎖定的源資料集。

C1鎖定 dataset · 計數稽核
C2重建 PRISMA · checklist
C3同一 script 重跑分析
C4清理 high-risk claims
C5投稿包同步稽核
幫我修稿 稿件 ready 了嗎 C1 dataset lock 投稿包準備 claim 清理
8
Deliverables · 輔助產出

最終交付檔案

公用

產出投稿檔案時搭配使用的公用 skill：Word 稿件與 Track Changes、資料集與統計表、轉檔表單、簡報。

docx — Word 稿件 / 追蹤修訂 xlsx — 資料集 / 統計表 pdf — 轉檔 / 表單 pptx — 研究簡報

本頁僅整理與研究工作流相關的 skill。帳號內另掛載 sales／marketing／finance／data／design 等商業領域 plugin，與 SR／統合分析無關，故未列入。
